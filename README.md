# Web Tools
Eine Sammlung von praktischen, clientseitigen Web-Tools für den alltäglichen Gebrauch. Möglichst viele Tools laufen direkt im Browser und benötigen keine Installation.
Sie sind auf meiner Github pages seite zu finden:
https://natanael207.github.io/CodedTools/

## Features & Vorteile
So weit wie möglich wird alles Lokal ausgeführt und benötigt so keinen Server.
Vorteil davon ist die Geschwindigkeit und dass die Daten sicher auf dem Gerät bleiben.

### Ausnahme
Bisher ist die einzige Ausnahme das WebRTC Tool. Es benötigt einen Firebase server für den Peer2Peer Verbindungsaufbau.
