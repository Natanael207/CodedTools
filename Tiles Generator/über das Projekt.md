# Tiles
Man kann ein Tile benutzen, um z.B. in einem Pixel game einen Sandigen Boden zu kreieren.
Mit dem Tile-Generator kann man Tiles random Generieren lassen. Man kann die Farben einstellen und verschiedene Muster wählen. Man kann die Tile-Grösse festlegen oder Symetrien auswählen.

# Blob-Generator
Blob-Generatoren können ein ganzes Tilesset kreieren. Ein Tilesset kann man benutzen, um übergänge zwischen verschiedenen Tiles arten zu machen. Dann hat man sogleich alle möglichen varianten von den Übergängen.
